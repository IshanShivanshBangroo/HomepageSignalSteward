# Signal Steward landing page

A standalone landing page for the Signal, Edit, Support activity.

## Files

- `index.html` — page markup
- `styles.css` — visual design
- `script.js` — lens switching, cursor glow, click bursts, and magnetic hover behavior

## External routes already linked in the page

- Participant view: `https://signalsteward.pages.dev/`
- Facilitator view: `https://signalsteward.pages.dev/facilitator`
